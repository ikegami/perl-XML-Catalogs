XML-Catalogs

XML::Catalogs provides a basic framework for locating
catalog of DTDs. XML parsers can use catalogs to avoid
downloading remote DTDs to parse and validate documents.


INSTALLATION

To install this module, run the following commands:

   perl Build.PL
   ./Build
   ./Build test
   ./Build install


DEPENDENCIES

This module requires these other modules and libraries:

    Module::Build    (For installation only)
    Test::More       (For testing only)
    Carp
    File::ShareDir
    URI::file
    version
